# 브랜드 로고 이미지 (개별 파일, 170×75)

각 브랜드별 로고를 **동일한 크기 170×75 px** 로 이 폴더에 넣어 두면, 양식·PDF 헤더에 표시됩니다.

## 파일명 (config.js `BRAND_LOGO_IMAGE` 와 일치)

| 브랜드 코드 | 브랜드명   | 파일명              |
|------------|------------|---------------------|
| HK10       | LANEIGE    | `logo_laneige.png`  |
| HK20       | Sulwhasoo  | `logo_sulwhasoo.png`|
| HK30       | AP         | `logo_ap.png`        |

- **저장 위치**: 이 README와 **같은 폴더** (`02.화면설계서/assets/images/`)
- **권장 크기**: **170×75 px** (가로×세로) — 코드에서 이 크기로 표시합니다.
- 파일이 없거나 로드 실패 시 브랜드명 텍스트(LANEIGE / Sulwhasoo / AP)로 자동 대체됩니다.
